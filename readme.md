**Laravel Role & Permission**
 - Laravel Role & Permission Using [Spatie](https://github.com/spatie/laravel-permission) . 
  
 - I have used  [PurpleAdmin](https://github.com/BootstrapDash/PurpleAdmin-Free-Admin-Template) for front end design .

**Step to run project**

 1. Clone this project using git ( `git clone https://github.com/vijendra3582/laravel-role-permission.git` ) or download this project and extract to a folder.
 2. Open project in terminal and run command `composer install` .
 3. Update .env.example  to .env .
 4. Change database credential & host details ( if not using localhost ) .
 5. Run command `php artisan key:generate`.
 6. Run command `php artisan migrate`  or upload sql file to database directly.
 7. Run command `php artisan db:seed`.
